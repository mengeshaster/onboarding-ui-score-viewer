# Development Scripts

Start development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Start production server:
```bash
npm run start
```

Run linting:
```bash
npm run lint
```

Run tests:
```bash
npm run test
```

Run tests in watch mode:
```bash
npm run test:watch
```

Format code:
```bash
npm run format
```

Generate OpenAPI types:
```bash
npm run openapi:gen
```
